# 🌤️ Weather Scanner Bot

Telegram бот для получения текущей погоды через WeatherAPI.com.

## 🚀 Быстрый старт

### 1. Получите API ключи
- Telegram: напишите [@BotFather](https://t.me/BotFather), создайте бота и получите токен
- WeatherAPI: зарегистрируйтесь на [weatherapi.com](https://www.weatherapi.com/signup.aspx), получите API ключ

### 2. Локальный запуск
```bash
export TELEGRAM_TOKEN="ваш_токен"
export WEATHER_API_KEY="ваш_ключ"
go run cmd/bot/main.go# test change
